
# Restaurant Web Application

This project is a web application for a restaurant. It includes features for user authentication, a dynamic menu display, a blog section, and information about the restaurant. The application is built using HTML, CSS (with Tailwind CSS for styling), and JavaScript.

## Table of Contents

- [How to Use](#how-to-use)

## Features
- **Menu:**
  - Dynamic menu display with categories.
  - Each menu item includes detailed information.
 - **Blog:**
   - A blog section for sharing updates or news about the restaurant.
- **User Authentication:**
  - Login and signup functionality.
  - Secure password handling and storage.
- **Responsive Design:**
  - The application is designed to be responsive and accessible on various devices.


## Technologies Used

- HTML
- CSS (Tailwind CSS)
- JavaScript

## How to Use

To get started with this project, follow these steps:

1. Clone the repository: `git clone https://github.com/WaqasAhmad203/Restaurant-Web-Application.git`
2. Open this project in VS Code 
3. Run this command `npm run start` in terminal
2. Then click on the local host link in terminal
